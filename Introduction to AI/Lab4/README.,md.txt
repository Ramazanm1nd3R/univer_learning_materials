# Отчёт по лабораторной работе: Анализ данных о образовании и бедности

## Введение
Цель данноЙ работы состоит в анализе данных об образовательном уровне и бедности в различных регионах. Работа направлена на выявление зависимостей между уровнем образования после 9 класса и уровнем бедности.

## Источники данных
Данные для анализа были взяты из официальной статистики регионов, включающей следующие параметры:
- Количество учащихся, окончивших 11 класс.
- Количество учащихся, окончивших 9 класс.
- Процент учащихся, продолживших обучение после 9 класса.
- Уровень бедности по регионам.

## Предварительная обработка данных
В ходе предварительной обработки данных были выполнены следующие шаги:
1. **Обработка пропущенных значений**: Пропущенные значения в столбцах были заменены на медианное значение для устранения искажений в данных.
2. **Нормализация данных**: Данные о количестве учащихся были нормализованы для обеспечения сопоставимости.
3. **Кодирование категориальных данных**: Названия регионов были преобразованы в числовой формат для анализа.

## Анализ корреляции
Был проведён корреляционный анализ между следующими переменными:
- Количество учащихся, окончивших 11 класс.
